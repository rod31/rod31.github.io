Litema

Cross-media recommender for movies, TV, and books that learns taste by vibe — mood, tone, pace, themes — not genre. Dual 1536-dim embeddings (vibe + plot) over pgvector ANN search, blended with Bayesian-smoothed ratings so obscure gems can surface alongside classics. Rating-weighted k-means on each user's library generates a personalized "For You" feed; batch GPT extraction via OpenAI's Batch API keeps ingestion cheap across 8,500+ titles. Python backend, React Native app, live at litema.app.

Python pgvector OpenAI Batch API Supabase k-means React Native TMDB API
